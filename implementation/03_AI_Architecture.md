# AI Architecture

AI roles:

Custodian:
- User-facing conversational guide
- Explains the archive
- Retrieves evidence
- Helps classify contradictions
- Can log new observations, claims, concepts, and tasks

Librarians:
- Claim Librarian
- Concept Librarian
- Revision Librarian
- Contradiction Librarian
- Planetarium Librarian
- Janitor Librarian

Provider policy:
- Provider-agnostic architecture
- Single active provider at runtime
- No accidental multi-provider billing
