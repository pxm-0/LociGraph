# Domain Model

Primary pipeline:

Source
-> Fragment
-> Observation
-> Claim
-> Concept
-> Revision
-> Graph Node
-> Planetary Object

Supporting layers:
- Reality Assertions
- Perception Assertions
- Interpretations
- Contradictions
- Importance Signals
- Custodian Logs

The Concept is the primary unit of the system.
Observations provide evidence.
Assertions separate factual claims from perceptual or interpretive claims.
