# LociGraph Vision

LociGraph is a concept-centric memory architecture.

It transforms raw personal data into created concepts, evidence-bound assertions,
interpretive models, contradictions, revisions, and planetary navigation structures.

Core principles:
- Concepts are primary.
- Reality and perception are separate.
- Evidence precedes interpretation.
- Contradictions remain unresolved unless reviewed.
- Concepts are created, not discovered.
- Raw sources are purgeable after verification.
- The primary interface is a planetarium.
- Custodian conversations can become logged observations.
