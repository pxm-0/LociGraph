# Planetarium Physics

Planet = Concept
Mass = Importance
Orbit = Evolution
Brightness = Activity
Moon = Subconcept
Ring = Contradiction
Constellation = Concept Cluster
Black Hole = Foundational Concept

Storage Layers:

Layer 1: Canonical storage
- Concepts
- Claims
- Observations
- Assertions
- Revisions

Layer 2: Semantic vector space
- Embeddings
- Model versions
- Vector metadata

Layer 3: Planetarium projection
- x, y, z
- radius
- theta
- phi
- projection version

The sphere is visualization, not truth.
The database is the canonical source of record.
